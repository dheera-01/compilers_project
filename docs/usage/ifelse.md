## Syntax for If-Elif-Else Statements

The code snippet provided illustrates the syntax for an if-elif-else statement in our programming language. The basic structure of the statement is as follows:

Input:
```python
assign i = 5;
if(i>5)
{
    print("inside if");
} 
elif(i == 4)
{
    print("elif 1");
}
elif(i == 5)
{
    print("elif 2");
}
else
{
    print("else");
}
```

Output:
`elif 2`
